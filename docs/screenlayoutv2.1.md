## 📐 USER FLOWS

### Flow 1: New User Onboarding
```
1. Splash Screen
   ↓
2. Sign Up (Email/Google/Apple)
   ↓
3. Birth Details Input
   ↓
4. Preferences Selection
   ↓
5. First Chart Generation (Loading)
   ↓
6. Welcome + First Daily Reading
   ↓
7. Guided Tour (Optional)
   ↓
8. Home Dashboard
```

### Flow 2: Daily Reading Consumption
```
1. Push Notification "Your reading is ready!"
   ↓
2. User opens app → Home screen
   ↓
3. Daily reading card visible (collapsed)
   ↓
4. User taps "Expand Full Reading"
   ↓
5. Reading expands with tabs (Work/Love/Health/Finance)
   ↓
6. User switches between tabs
   ↓
7. User taps "Ask AI" for follow-up questions
   ↓
8. Opens chat with pre-filled context
```

### Flow 3: AI Chat Interaction
```
1. User taps chat icon in bottom nav
   ↓
2. Chat interface opens
   ↓
3. User types question
   ↓
4. AI responds with text + chart references
   ↓
5. User taps chart reference → opens detailed view
   ↓
6. User asks follow-up question
   ↓
7. Conversation continues
   ↓
8. (If quota exceeded) → Upgrade modal shown
```

### Flow 4: Compatibility Check
```
1. User taps "Match" quick action
   ↓
2. Select or add partner profile
   ↓
3. Enter partner's birth details
   ↓
4. Loading animation (calculating)
   ↓
5. Results screen with score gauge
   ↓
6. Scroll to see strengths & watch-outs
   ↓
7. Option to save, share, or get full report
   ↓
8. (If Pro) → Unlock detailed compatibility PDF
```

### Flow 5: Admin User Management
```
1. Admin logs in with OTP
   ↓
2. Dashboard overview
   ↓
3. Clicks "Users" in navigation
   ↓
4. User table loads with search/filter
   ↓
5. Admin searches for specific user
   ↓
6. Clicks user row → detail modal opens
   ↓
7. Reviews usage stats and entitlements
   ↓
8. Edits entitlements (increase quota)
   ↓
9. Saves changes → audit log created
   ↓
10. Modal closes → table updates
```
